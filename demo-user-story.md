# 《小墨 AI 学习交互系统》Demo Storyboard  
## 主线：考前 4–12 小时冲刺复习（单一 User Story）  
## 必含爆点：① 个性化老师（RAG） ② 5 小时冲刺路径（系统级方案）

---

# Scene 1：开场 —— 考前焦虑、资料混乱

**画面：**  
桌面杂乱堆满文件（PDF/PPT/JPG/Word）：  
- Lecture slides（40+）  
- Past exams（PDF）  
- Scanned homework  
- 同学的复习笔记  
- 老师发的复习纲要  

用户不断切文件、关文件、再切文件。  
右下角出现倒计时：「考试倒计时：5 小时 12 分」。

**用户操作：**  
快速点击多个 PDF → 反复打开 → 表情焦虑。

**系统台词（旁白）：**  
「考前 5 小时，你要面对 400+ 页资料，不知道重点、不知道题型、不知道从哪里开始。」

**投资人看到：**  
- 大学生真实场景  
- “混乱 → 系统介入” 戏剧结构准备完成  
- 这是 ChatGPT 无法处理的场景

---

# Scene 2：打开小墨 —— 平静、秩序、可依赖

**画面：**  
干净界面，首页中央大字：  
「把所有考试资料拖进来，小墨会帮你生成冲刺方案。」

左侧侧边栏为空（未添加科目）。  
右侧一个大框提示拖拽区。

**用户操作：**  
将所有杂乱文件 → 一次性拖拽到小墨。

**小墨反应：**  
自动识别文件类型并展示分类标签：  
- Lecture 1 Slides  
- Past Exam 2021  
- Homework Set  
- Practice Solutions  
- Textbook Ch.3  

动态亮起，一条条进入系统。

**投资人看到：**  
- 小墨是系统，不是聊天框  
- “一次拖拽 → 全部理解” 远胜 GPT-PDF 总结  
- 开始建立信任感

---

# Scene 3：资料自动结构解析（爆点前戏）

**画面：**  
屏幕中央生成一张“资料结构图”：  
节点包括：  
- Key Concepts  
- Formula Groups  
- High-Frequency Exam Topics  
- Common Mistakes  
- Important Examples  

节点自动扩散 → 折叠成整洁的“考点地图”。

**系统台词：**  
「你的资料我已经读完了。下面我需要确认一些你的个人情况。」

**用户操作：**  
点击“继续”。

**投资人看到：**  
- 小墨不仅能分类，还能理解内容  
- 强烈的戏剧感，准备进入爆点 1  
- 这是难度极高的“系统级 RAG 能力”

---

# Scene 4：爆点 1 —— 个性化老师（RAG）

**画面：**  
小墨弹出 3 张关键个性化问答卡片：

1）「你对本门课的掌握程度？」  
滑杆：0%–100% → 用户选择“40%”

2）「你的弱项是什么？」  
多选：推导题 / 概念题 / 应用题 → 选“推导题”

3）「你今天有多少有效学习时间？」  
4/5/6 小时 → 选“5 小时”

问完后卡片被吸入中央 → 生成“用户画像 × 内容分析”的动态融合动画。

**小墨台词（关键）：**  
「我已经了解你的当前状态，我会为你生成一条 **只属于你** 的 5 小时冲刺路径。」

**投资人看到：**  
- 真正的个性化，不是用户自己编问题  
- 小墨“懂你 + 懂资料”  
- 这是 Demo 第一个核心爆点  
- GPT 不能做到这种“系统级个性化”

---

# Scene 5：爆点 2 —— 生成 5 小时冲刺路线图（系统级学习方案）

**画面（最戏剧化的一幕）：**  
中央动态出现一条时间轴（Timeline）：

```

🍊 第 0:00 — 0:40
高频概念 + 你最弱的推导题核心逻辑

🔥 第 0:40 — 1:20
往年推导题精练（根据你之前错题挑选）

📘 第 1:20 — 2:00
高频例题（过去 5 年出现 5 次）

🧩 第 2:00 — 3:00
综合练习 + 自动检测你的掌握度

🌕 第 3:00 — 4:20
模拟考试（根据你弱点动态生成）

✨ 第 4:20 — 5:00
最终查漏补缺（因你而变的个性化清单）

```

时间片段依次亮起并连成连贯路径。

**小墨台词：**  
「这是你今天的最佳冲刺方案。在 5 小时内，我会确保你覆盖全部必考内容，并补齐你的弱项。」

**投资人看到：**  
- 真正的系统（路径图 = 不是聊天）  
- 立刻能理解“这比 ChatGPT 强太多”  
- 二次爆点：整体解决方案  
- 市面上无人做到  
- 这就是“学习操作系统”的雏形

# Scene 5.1：概念复习（0:00 — 0:40）

**画面：**  
顶部固定时间轴（高亮 0:00–0:40），右上角倒计时「40:00」。  
左栏「高频概念清单」按【高频/易错/未掌握】三种标签聚合，每个概念带小圆环掌握度与出现频次。  
中间为「概念速记卡」：要点 3 条 + 关键公式 + 典型误解，卡片底部有 20s 微测验条。  
右栏为「原文定位区」：多源 PDF 定位跳转，高亮与概念锚点同步滚动。

**用户操作：**  
点击筛选【未掌握】→ 点选「线性估计的偏差与方差」概念 → 向下滑动查看误区示例 → 勾选「记住了」→ 进行 3 题微测验（判断/填空）→ 键盘 N 跳到下一个概念。  
在公式处长按复制 → 拖拽到右侧「草稿区」标注。

**AI 反馈：**  
入场即提示「你的资料中此概念近 5 年出现 4 次，与你的错题关联度 87%」。  
测验中即时判分并标出错误模式：「你将充分条件当成必要条件，建议回看第 2 条要点」。  
若同一概念连续答对 3 次：自动压缩讲解为「10s 复述卡」。  
若停留 > 90s：弹出「这部分为低收益内容，建议跳过 2/5」。

**信息变化：**  
「概念掌握度」环形进度从 40% → 58% → 63%。  
「覆盖概念数」3 → 7。  
倒计时从 40:00 减至 31:10；右上角出现「节奏良好」绿标。  
预测得分 +3（来自高频概念）。

**节奏：**  
快节奏卡片滚动，测验正确时卡片底部短促绿光扫过；转场采用「卡片折叠 → 下个概念弹入」。  
10 分钟节点轻震提醒一次，强调时间压力。

**目的：**  
证明小墨「懂资料」（高频/误区/来源映射）与「懂用户」（误判类型与讲解粒度自适应），建立第一段可见的学习增益。

---

# Scene 5.2：往年题训练（0:40 — 1:20）

**画面：**  
左侧「题库筛选区」按【年份/难度/考点】切片，默认勾选「推导题」「高频」。  
中间为「分步作答板」：步骤格子 1→2→3，每格可输入公式/文字，支持逐步校验。  
右侧抽屉为「公式库 × 思路模板」，含常用推导套路与可拖拽的结论块。

**用户操作：**  
选中 2021 Q3 与 2023 Q2 两题 → 点击「开始训练」。  
在步骤 1 输入先验设定 → 步骤 2 卡住 → 点击「看提示 1/3」→ 拖拽「正态近似条件」到作答板 → 继续输入。  
若仍卡住 → 点击「换一问（保留未完）」。

**AI 反馈：**  
进题前给出「预计耗时 7–9 分钟」「易错点：边界条件」提示。  
逐步校验：在步骤 2 标出红色高亮「缺少约束项」，旁注「你在作业 HW4 也错在此处」。  
若耗时超出 120%：自动降维提示「先证明引理 A」，并推荐 90s 微例题热启动。  
完成后给「错误模式」标签：边界忽视 / 推导跳步 / 条件混淆。

**信息变化：**  
本阶段正确率 62% → 78%；平均每题用时 9:20 → 6:10。  
错因计数（边界忽视）2 → 1；弱项权重从 0.7 降至 0.55。  
时间条从 40:00 → 19:30，显示「略慢，已自动压缩下一题讲解 20%」。

**节奏：**  
每题形成小闭环：入场预测 → 分步作答 → 局部提示 → 纠错收束。  
转场使用「题卡翻页」动效，正确时绿勾弹出，错误时红点标注错因。

**目的：**  
展示小墨将「你的历史错误 × 往年题结构」动态重组，题目选择与提示强个性化，体现“懂用户”的难模仿性。

# Scene 5.3：例题突破（1:20 — 2:00）

**画面：**  
中间「关键例题讲解卡」分为「题干」「思路」「关键跳步」「易错对比」。  
左侧出现「你的作答 vs 标准解」对照板，差异处高亮成黄色热区。  
右侧为「一步一提示」滑轨，最多 3 级，默认折叠。

**用户操作：**  
点击「隐藏答案」→ 只看题干先作答 → 在草稿区写下第一步 → 点击「对比」查看差异 → 打开提示 1/3。  
选择「我要更难/更易」调节器 → 切换同考点不同难度例题。  
将「关键跳步」收藏进「易错片段」清单。

**AI 反馈：**  
预测下一步成功率 54%，给出「提醒：先统一符号」的最小化提示，不直接给结论。  
若两次提示后仍错误：自动降级为「引导式填空」，并显示「你在概念复习中掌握了 X，可直接调用」。  
例题完成后回写到「概念掌握」图谱，提示「此例题巩固了‘充分必要’概念」。

**信息变化：**  
该考点掌握度 63% → 74%；下一步成功率在一题内动态上升 54% → 81%。  
个人「关键跳步」库 +2 条。  
时间剩余 22:40 → 3:10（本阶段）。

**节奏：**  
以「揭示式」节奏推进：每展开一层提示，卡片轻微下沉与发光，强化推进感。  
完成时例题卡折叠成「一页讲义」留存。

**目的：**  
让投资人看到：小墨把静态讲解变成「可控粒度的交互剧本」，既不“直接公布答案”，又高效拉升步骤正确率。

---

# Scene 5.4：综合练习（2:00 — 3:00）

**画面：**  
题组卡（10–12 题）混合单选/填空/推导小问，每题角标标识考点与预计用时。  
顶部出现「策略滑杆」：速度优先 ↔ 准确优先，默认中间。  
右侧显示「掌握雷达图」：概念/推导/应用/计算 四维实时刷新。

**用户操作：**  
按快捷键 A/S/D 快速作答与跳题，遇到不熟的标记「没把握」。  
将策略滑杆向「速度优先」拨动 20%，以适配时间压力。  
做完 5 题后点「小结」，查看错因分布。

**AI 反馈：**  
根据滑杆即时调整题型比例：推导题 -20%，应用题 +15%。  
错因分类器将错误归入「读题遗漏」「单位不一致」「边界忽视」，并给出每类 30s 纠偏卡。  
检测到对「边界忽视」有进步后，恢复部分推导题占比，保持挑战度。  
若 5 分钟内 3 连错：发出「节奏保护」提示，建议跳过 1 题并回收 90s。

**信息变化：**  
阶段正确率 76% → 83%；速度评分 +12。  
雷达图推导维度从 0.55 → 0.62；应用维度 0.68 → 0.75。  
预测得分区间 71±6 → 74±5。

**节奏：**  
中速推进，每 10 分钟出现一次「小结弹层」；题卡翻转与雷达图脉冲同步，制造节拍感。  
时间条在 5 分钟与 1 分钟处各有一次轻震提醒。

**目的：**  
体现小墨的「系统级编排」：根据策略、表现与剩余时间实时重配题型与难度，保证学习效用最大化。

# Scene 5.5：自适应模拟（3:00 — 4:20）

**画面：**  
全屏「考试模式」：顶部总时长 80:00，左侧题号导航，右侧草稿区与公式速查，提示默认关闭。  
状态栏低调显示「配速：正常 / 偏慢 / 偏快」。

**用户操作：**  
按题号顺序作答，标记第 6 题为「稍后」，对第 8 题开启「草稿网格」。  
遇到计算量大题目使用「一键估算」检查数量级。  
40 分钟时点击「进度盘」查看完成度与耗时分布，再返回作答。

**AI 反馈：**  
实时配速监控：若连续 3 题超时，发出黄标「节奏警报」并建议跳过一道高耗时题。  
动态削峰：检测到你在推导题稳定正确后，后半段自动替换 1 题为更具区分度的应用题。  
提交前 5 分钟，给出「最划算的 3 分」建议，指向两道短题与一道填空。

**信息变化：**  
完成度从 0% → 72% → 94%；剩余时间 80:00 → 05:00。  
预测得分区间从 72±7 收窄到 77±4。  
跳题池 3 → 1。

**节奏：**  
稳定推进，配速条以呼吸灯节奏变化；当触发「节奏警报」时短促红闪 0.5s，随后回到平稳。  
交卷时使用「纸张合拢」动画收束。

**目的：**  
让投资人直观看到小墨不仅能出题，还能「实时调度难度与收益」，把模拟从静态试卷升级为动态系统。

# Scene 5.6：查漏补缺（4:20 — 5:00）

**画面：**  
「错因清单」按收益排序自动展开：立刻可补的 6 分、容易翻车的 4 分、低收益内容收起。  
右侧为「一屏电梯讲义」：每个错因配 30–45s 微讲解 + 2 题微测验。  
底部有「考前 10 分钟提醒」开关与「保存我的路线」按钮。

**用户操作：**  
依序点选清单条目 → 观看 30s 动画讲解 → 立即做 2 题微测 → 点亮绿勾。  
对仍不稳的条目点击「加入考前提醒」。  
点击「保存我的路线」并导出 PDF 讲义。

**AI 反馈：**  
自动重排查漏顺序：优先推送「高分且短时可修复」的项。  
对连续两次微测正确的条目标注「已复原」，并从清单中折叠。  
生成个性化「考前 10 分钟冲刺卡」并设置提醒。

**信息变化：**  
剩余清单 9 → 3 → 0；复原率 0.0 → 0.82 → 1.0。  
预测得分从 77±4 → 80±3。  
倒计时从 40:00 → 00:00（收束）。

**节奏：**  
快节奏「打勾」反馈与烟花小动效；每清一项，路线上相应节点点亮。  
最后一项完成触发「完成徽章」与轻音乐收尾。

**目的：**  
突出「闭环」：从资料理解 → 路线编排 → 训练 → 模拟 → 查漏 → 导出与提醒。让投资人看到可交付的结果与复用价值。

# Scene 7：冲刺结束 —— 总结 + 个性化评估（闭环）

**画面：**  
左侧「成果总览」：覆盖考点、错因复原率、弱项提升、预测得分 80 ± 3。  
中间「阶段回放（5.1–5.6）」小时间轴，节点显示每段提分贡献与耗时占比。  
右侧「错因拆解」：边界忽视 / 跳步 / 条件混淆，对应纠偏卡与通过率。

**用户操作：**  
滚动查看阶段回放 → 展开「提分来源」分解 → 点击「生成二刷路线」。  
打开「考前 10 分钟冲刺卡」预览 → 勾选短信/日历提醒。  
点击「保存我的路线」并导出一页讲义 / PDF。

**AI 反馈：**  
汇总提分分解（示例）：概念 +3、往年题 +2、自适应模拟 +2、查漏 +1。  
解释置信区间收窄：题量×一致性×错因复原度提升。  
生成「二刷加速路线」（压缩至 90 分钟）与「考场策略建议」。

**信息变化：**  
预测得分 77±4 → 80±3；弱项权重 0.55 → 0.38；复原率 1.0（已清零清单）。  
路线状态：已保存；提醒：已设置；知识库新增「关键跳步 ×2」「纠偏卡 ×3」。

**节奏：**  
数字轻微跳动填充；时间轴节点按 5.1 → 5.6 依次点亮。  
导出触发「打包进度」线性条，2 秒内完成。

**目的：**  
向学习者与投资人证明闭环与可复用性：清晰的提分来源、可复制的个人路线、明确的考前执行物。

# Scene 8：结尾 —— 投资人 Takeaway（价值锚定）

**画面：**  
黑底白字，三张亮点卡片横排：  
– 系统级 RAG：跨多源资料自动建模与锚点定位。  
– 自适应路径：按时间与掌握度动态调度内容与难度。  
– 实时决策：配速/错因/收益优先级联动。  
底部定位句：  
「小墨 = 懂资料 × 懂用户 × 给复习路线 的 AI 学习系统」  
「在最短时间内，把最该学的东西，教给你。」

**用户操作：**  
点击「30s 亮点回放」或「切换另一门课快速演示」。

**AI 反馈：**  
提示「该 Demo 可接入 API / LMS」并显示集成标签（Canvas / Moodle / 钉钉）。

**信息变化：**  
转化线索：保存路线 1 次、提醒开启 1 次、导出讲义 1 份。

**节奏：**  
亮点卡片依次放大出现，定位句最后落版；全片淡出收尾。

**目的：**  
对投资人锚定产品差异化与可规模化能力：不是聊天，而是系统级「学习操作系统」。
