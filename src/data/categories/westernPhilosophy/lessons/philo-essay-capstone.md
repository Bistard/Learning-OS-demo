# 哲学写作冲刺

## 写作步骤
1. **选题**：
   - 题 A：存在主义如何理解自由？
   - 题 B：批判理论为何强调文化工业？
2. **列提纲**：写出论点、论据、案例。
3. **写初稿**：保持 800 字左右，段落结构：引言 → 论证 → 反驳 → 结论。
4. **AI 批注**：请 RAG 老师标出逻辑跳跃或概念混乱之处。
5. **终稿**：根据批注修改，并在结尾加一句“未来拓展”。

## 评分标准
- 观点清晰（30%）
- 论据充分（30%）
- 结构完整（20%）
- 语言与引用准确（20%）

> 完成后，将终稿复制到知识库「论证拆解」，方便下一次复用。
