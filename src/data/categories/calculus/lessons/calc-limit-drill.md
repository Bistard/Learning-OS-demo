# 极限典型题演练

## 题单
1. $lim_{x→0} (1-\cos x)/x^2$
2. $lim_{x→0} \frac{\sin 3x}{\sin 2x}$
3. $lim_{x→0} x^x$

## 操作提示
- **步骤 1：识别类型**。写下“夹逼 / 等价无穷小 / 洛必达”。
- **步骤 2：列中间量**。例如 $1-\cos x ≈ x^2/2$，或 $x^x = e^{x \ln x}$。
- **步骤 3：AI 复核**。把推导输入 RAG 老师，让其指出遗漏的条件。

## 练测循环
| 题目 | 首次得分 | 第二次 | 要点 |
| ---- | ---- | ---- | ---- |
| 1 | 2/3 | 3/3 | 忘记写“当 x→0 时”|
| 2 | 1/3 | 3/3 | 倒数错写成 $2/3$|
| 3 | 0/3 | 2/3 | 忽视 $x \ln x → 0$|

> 反思：若第二次仍不过关，把题干加入 Mistakes Vault，并要求 AI 生成同型变式。
