# 积分 checkpoint 测验

> 限时 15 分钟，完成 5 道题，提交后 AI 自动判分并生成错因报告。

## 测验结构
1. 基础：换元/分部各 1 题。
2. 综合：面积或旋转体 2 题。
3. 反常积分 1 题。

## 评分与复盘
- 80 分以上：可直接进入下一任务。
- 60~79 分：根据错题生成抽认卡，24 小时后复测。
- 60 分以下：回到 `calc-integral-sprint`，重做同型练习。

## 记录模板
```
| 题号 | 得分 | 错因 | 下一步 |
| ---- | ---- | ---- | ---- |
| 1 | 1/1 | - | 继续 |
| 3 | 0/2 | 面积公式写错 | 复习柱/盘法 |
```

完成后点击「同步到知识库」，会自动创建一条 Mistake 记录。
