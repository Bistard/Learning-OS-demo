# 积分 checkpoint：仿真 30 分钟考场

## 该任务的详细学习内容
### Priority Learning（检测点）
- 三段式卷面：
  1. **换元速算**（5 分）
  2. **定积分面积/体积**（10 分）
  3. **应用题综合**（15 分）
- 为每段写“得分底线”：先保证步骤完整，再追求代数精准。

### Chunking 拆解步骤（应试流程）
1. **读题标注**：划出积分区间、已知条件、所求量。
2. **方法预判**：在题旁写下“换元 / 分部 / 几何”标签，防止做一半才想起更优策略。
3. **限时演练**：每小题设闹钟，严格按 10-12-8 分钟分配。
4. **查漏**：做完立即对照关键步骤清单，补写理由句避免被扣“格式分”。

### Feynman Technique
- 把自己的解题草稿二次讲给“未来的自己”：用 1 分钟复盘“我为什么选这个方法”，AI 检查逻辑。

### Prediction-Based Learning
- 根据往年卷，定积分综合压轴常涉及“旋转体或弧长 + 参数”。提前准备“模板答题卡”：
  - 写出体积公式
  - 列积分
  - 换回原变量并解释实际意义
- 把这套模板练到闭卷可默写。

### Active Recall（快速回忆）
1. 说出计算旋转体体积的两种常用公式。
2. 口述检查定积分答案时的三个观察点。
3. 用一句话解释换元后上下限如何变化。

### Quiz Generation（仿真题）
1. $I_1=\int_{0}^{1} \frac{1}{\sqrt{1+x^2}} dx$（换元题）
2. 由 $y=x^2$、$y=4x$ 以及 $x=0$ 围成区域绕 $y$ 轴旋转，求体积。
3. 水箱注水速率满足 $\frac{dV}{dt}=3t+2$，已知 $V(0)=0$，求 $V(2)$ 并说明物理意义。

## 为什么学这个？（优先级理由）
- 考前 24 小时必须经历至少一次完整模拟，才能把零散技巧固化为考场节奏。
- 通过限时检测，你能定位“步骤漏写”“公式忘记”这类低级失分点，及时补救。
- 成绩区分度往往就在这 20~30 分，练熟即可拉开差距。
