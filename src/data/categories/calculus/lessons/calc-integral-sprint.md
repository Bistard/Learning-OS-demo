# 5 小时积分突击：面积与原函数

## 该任务的详细学习内容
### Priority Learning（必考清单）
- **不定积分模板**：指数、三角、反三角、分式四大类，列出匹配信号（见幂次/根号/指数的组合）。
- **定积分几何意义**：把“面积=上-下”写成一句话：$S=\int_a^b (f(x)-g(x))\,dx$，理解“符号=方向”。
- **换元 vs. 分部**：写出“先观察嵌套关系再选法”的判据，杜绝盲目试错。

### Chunking 拆解步骤（以换元为例）
1. **锁定内层**：找最复杂的嵌套表达式设为 $u$。
2. **求导替换**：计算 $du$，整理出与原积分中同款的微分。
3. **改写积分**：完全改成 $u$ 的世界，定积分记得同步改上下限。
4. **回代解释**：得到原变量表达式后，再复述“面积/物理意义”。

### Feynman Technique
- 用语音向 AI 解释“为什么定积分的符号可以表示方向”，AI 若追问，就用数轴故事再讲一遍。
- 尝试把“分部积分”讲成“积分版乘法公式”：写下“谁来当 $u$，谁来当 $dv$，为什么”。

### Prediction-Based Learning
- 预测大题结构：
  1. 先求原函数或定积分。
  2. 再把结果带进面积/体积/旋转体模型。
  3. 最后考查积分与导数混合（如微分方程分离变量）。
- 针对卷面惯例，准备“旋转体三步曲”：平移坐标、写半径、套 $\pi \int r^2 dx$ 或 $2\pi \int r\cdot h dx$。

### Active Recall（快速回忆）
1. 说出分部积分公式并解释“谁是 $u$，谁是 $dv$”。
2. 口述定积分求面积时如何处理交点。
3. 回忆一个换元积分例子，从设元到回代全程复述。

### Quiz Generation（无需写全解）
1. 求 $\int x e^{x^2} \, dx$。
2. 计算 $\int_{0}^{\pi/2} \sin^3 x \, dx$。
3. 设区域由 $y=x^2$ 与 $y=2x$ 围成，绕 $x$ 轴旋转求体积。

## 为什么学这个？（优先级理由）
- 积分题在期末总分中占比近 30%，是冲线阶段最应该“限时突围”的板块。
- 从零开始也能靠模板拿分，重点是把抽象概念翻译成面积/体积的直观故事。
- 该任务紧接导数模块，帮助你在 5~8 小时冲刺里实现“从变化率到累积量”的完整闭环。
