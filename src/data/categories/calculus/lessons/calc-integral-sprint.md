# 定积分与面积体积

## 方法对照表
| 信号 | 建议方法 | 备注 |
| ---- | ---- | ---- |
| 含 $e^{ax}$ 或 $\sin bx$ | 分部积分 | 先令多项式为 u |
| 对称区间 $[-a,a]$ | 奇偶性/对称性 | 检查 integrand 奇偶 |
| 根式 + 一次式 | 换元 | 设置 $u = \sqrt{...}$ |

## 学习路径
1. **打法标签化**：看题马上写下 “换元/分部/凑微分/几何”。
2. **几何可视化**：对面积/体积题，手绘示意并写上单位。
3. **AI 复盘**：用自然语言解释「为什么能这么换元」，训练费曼复述能力。

## 典型例题
1. $\int x e^{x} dx$ —— 分部积分。
2. $\int_{-1}^{1} \frac{x}{\sqrt{1+x^2}} dx$ —— 奇函数，答案为 0。
3. 旋转体体积：$y=x^2$ 与 $y=2x$ 围成区域绕 x 轴旋转。
