# 导数与最值一条龙

## 结构化流程
1. **定义复盘**：写出导数极限定义、链式法则、隐函数求导。
2. **图像直觉**：用 Desmos/AI 绘出函数与导函数，标注单调与拐点。
3. **最值题模板**：
   - 求一阶导，列符号表。
   - 写出增减性文字说明。
   - 若为应用题，补上“量纲解释”。
4. **压轴拓展**：挑一题需要用到柯西中值或拉格朗日中值，记录“为什么想到它”。

## 例题仓
- 函数 $f(x)=x^3-3x+1$：
  - 求极值点与最大值。
  - 解释函数图像为何呈“S”型。
- 参数方程 $\begin{cases}x=t^2+1\\ y=t^3-3t\end{cases}$：
  - 求 dy/dx，并判断在 $(2,0)$ 附近的凹凸性。

## 课堂产出
```text
导数模板卡：
- 链式：外导×内导，必要时写出替换变量。
- 隐函数：两边同时对 x 求导，补上 dy/dx。
- 参函数：先对参数求导，再除以 dx/dt。
```
