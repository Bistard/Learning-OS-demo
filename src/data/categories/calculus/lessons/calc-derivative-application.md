# 从斜率到最值：导数应用冲刺

## 该任务的详细学习内容
### Priority Learning（常考场景）
- 三大题型：函数单调与极值、切线/法线方程、相关变化率。各自列出首个判别式：$f'(x)=0$、$y-y_0=f'(x_0)(x-x_0)$、$\frac{dy}{dt}=\frac{dy}{dx}\frac{dx}{dt}$。
- 15 分钟内读懂题干，将文字描述转成“待求参数 + 约束方程”列表，避免掉入题海。

### Chunking 拆解步骤（以最值题为例）
1. **建模**：整理题中变量、约束、目标函数。
2. **求导**：写清 $f'(x)$ 并因式分解，找驻点与定义域边界。
3. **判别**：用一阶导符号表或二阶导判断极值性质。
4. **回代解释**：带回原题语境，写出尺寸/时间/速度的物理意义。

### Feynman Technique
- 把“求函数的切线”讲给白纸：先给出曲线、点，再写导数公式，最后联立得到直线方程。录下讲解，AI 检查是否遗漏通过点。
- 若 AI 提醒错漏，立即补写“常犯错警示卡”，贴在讲义边上。

### Prediction-Based Learning
- 依据近三年试题，导数压轴常出现“应用题 + 限时 20 分钟”。提前练：
  - **模型组合**：几何最值（面积/体积）、经济最值（成本/收益）。
  - **微分方程式**：$y'=ky$ 或 $y'+py=q$ 的解析解，常作为附件小问。
- 预设考试情景：先写“变量表”，再写“导数关系”。练到闭眼都能复现。

### Active Recall（快速回忆）
1. 一句话解释如何判断 $f(x)$ 在区间上单调递增。
2. 口述求过点 $(x_0,y_0)$ 的法线方程的步骤。
3. 说出相关变化率题目中最容易漏掉的链式法则环节。

### Quiz Generation（无需写全解）
1. 已知 $f(x)=x^3-3x+1$，求其极值点并判断类型。
2. 曲线 $y=\ln x$ 在 $x=e$ 处的切线与法线方程。
3. 设 $x$、$y$ 满足 $x^2+y^2=25$，且 $\frac{dx}{dt}=3$，求 $x=4$ 时的 $\frac{dy}{dt}$。

## 为什么学这个？（优先级理由）
- 导数应用一题动辄 20 分，只要流程熟练就能“先拿稳 12 分，再冲附加题”。
- 讲练结合能让你真正理解导数的物理意义，后续积分与级数题阅读也更顺畅。
- 在 5~8 小时冲刺中，这是承上启下的 Part 3，帮助你从概念跨到综合。
